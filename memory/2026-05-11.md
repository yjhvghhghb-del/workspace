- 使用 curl：
  - 百度：HTTP/1.1 200 OK
  - Google：HTTP/2 200
  - YouTube：HTTP/2 200
  - GitHub：HTTP/2 200

**结论**：
1. 系统网络完全正常，能访问国内+国外所有网站
2. DNS 正常（使用 8.8.8.8）
3. web_fetch 工具有问题/限制，curl 完全可用

**经验**：
- 以后需要抓网页时，用 `curl` 代替 `web_fetch`
- curl 格式：`curl -I --connect-timeout 5 https://URL` 测试访问性
- 如果需要全文：`curl -s https://URL`
